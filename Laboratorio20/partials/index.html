Preguntas:

Explica y elabora un diagrama sobre cómo funciona AJAX con jQuery.





En AJAX la función $.ajax permite definir un objeto de datos para el jQuery. AJAX se comunica con el servidor para pedir los fragmentos que le faltan a la página. Estos después los transmite en la página con formato JSON.[1]

¿Qué alternativas a jQuery existen?

  • Javascript puro, sin librerías.
  • Zepto.js: Estructura modular dirigida a entornos muy exigentes en rendimiento.
  • XUI: Para el desarrollo de aplicaciones móviles
  • Sizzle: Rendimiento competitivo y es independiente.
  • Qwery: Simple y útil para seleccionar elementos en el DOM. [2]
