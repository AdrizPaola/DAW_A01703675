Preguntas

  • ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    • Diferencias:
    • Java es un lenguaje que esta orientado a objetos puro mientras que JavaScript esta basado en prototipos.
    • JavaScript no es un lenguaje compilado, este se integra directamente en las páginas HTML siendo interpretado por el navegador.
    • Java es más complejo ya que es un leguaje con más capacidades, por otro lado, JavaScript tiene menos capacidades y es más sencillo.
    • JavaScript es estándar, Java no.
    • Java necesita un kit de desarrollo y máquina virtual para poder trabajar, mientras que JavaScript solo necesita un editor de texto.
    • En Java necesitamos declarar las variables, por otro lado, en JavaScript podemos usar una variable de manera flexible.
    • Java es independiente de la plataforma con el código de bytes, mientras que en JavaScript necesita ser escrito de acuerdo con la compatibilidad del navegador.
    • No existen reglas para constructores en JavaScript como en Java.
    • JavaScript tiene mas popularidad con respecto a proyectos en el desarrollador de GitHub creados en los últimos 5 años. [1]

    Semejanzas:
    • Ambos tienen una sintaxis similar a C.
    • Ambos cuentan con una extensa documentación, foros en línea y comunidades en las que se puede aprender fácilmente.
    • Ambos tienen la capacidad de trabajar tanto en un buscador como en un servidor. [1]


  • ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
    • getDate(): Nos devuelve el día del mes, del 1 al 31.
    • toTimeString(): Convierte la parte de tiempo de un objeto Date en una cadena.
    • parse(): Analiza la fecha y devuelve el número de milisegundos transcurridos desde el 1 de enero de 1970 hasta la fecha analizada.
    • getDay(): Regresa el día de la semana. Entre 0 (domingo) y 6 (sábado).
    • getMinutes(): Devuelve los minutos, entre 0 y 59.
    • getMonth(): Devuelve el mes. Entre 0 (enero) y 11 (diciembre)
    • setDate(): Actualiza el día del mes. [2]



  • ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
    • lenght(): Para conocer el número de elementos del arreglo.
    • push(): Acepta cualquier número de argumentos y los agrega al final del arreglo.
    • join(): Junta todos los elementos de una lista separándolos por el carácter que se indique dentro del paréntesis.
    • pop(): Elimina el último elemento del arreglo y disminuye de su longitud una posición.
    • includes(): Determina si el array contiene un determinado elemento.
    • toString(): Para convertir un arreglo en una cadena. Mostrará todo el array en una cadena en la que cada elemento estará separado por comas. [3]



  • ¿Cómo se declara una variable con alcance local dentro de una función?
    Dentro de la función en la que se utilizará la variable se debe declarar la variable. Ejemplo:

    function crearMensaje() {
    var mensaje = "Hola";
    }

    Lo que significa que la variable solo podrá ser usada dentro de la función, es importante declararla con var, ya que de otro modo se volvería global. [4]


  • ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Las variables globales se convierten en atributos del objeto window y aunque la función termine esta no se eliminará sino hasta que la ventana sea cerrada. El uso de variables globales debe ser justificado con claridad. [5]


  • ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
    Se utilizaría el método match() , ya que este se usa para buscar coincidencias con cualquier expresión regular y en caso de que esta llegue a encontrarse entonces se devolverá como un arreglo, de no ser así retornará Null. En una página web se puede implementar en la búsqueda de contenido, pues le sería de gran utilidad al usuario por la facilidad de encontrar lo que busca solo ingresando palabras clave por asi decirlo.
